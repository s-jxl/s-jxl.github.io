Le modèle client-serveur

  • Dans le monde informatique, ce modèle est utilisé partout

  • Il est donc important de comprendre comment ça fonctionne, pour pouvoir commprendre le reste


    • Le client:

  • Gère l'aspect visual du site/application

  • Demande au serveur pour de l'information

  • Contient du code qui est visible

  • Représente votre appareil


    • Le serveur:

  • Sert des fichiers

  • Gère les bases de données

  • Répond ou ne répond pas aux demandes du client

  • Assure la sécurité

  • Ne contient pas de code visible

  • Représente un autre appareil, normallement situé dans le quartier général des compagnies


    • Le client établie une connexion à travers l'Internet avec le serveur
